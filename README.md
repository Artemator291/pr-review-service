# PR Review Assignment Service

C++ микросервис для автоматического назначения ревьюеров на Pull Request'ы.

## Функциональность

- ✅ Создание команд и управление пользователями
- ✅ Автоматическое назначение ревьюеров при создании PR
- ✅ Переназначение ревьюеров для открытых PR
- ✅ Мерж PR (идемпотентная операция)
- ✅ Получение списка PR для ревью пользователя
- ✅ Управление активностью пользователей

## Допущение
- Пользователь может быть только из одной команды

## Дополнительные функции

### Статистика
Эндпоинт `/stats/review-assignments` предоставляет:
- Общую статистику по PR
- Количество назначений по пользователям
- Распределение PR по статусам

### Массовая деактивация
Эндпоинт `POST /users/bulk-deactivate` позволяет:
- Деактивировать нескольких пользователей одновременно
- Автоматически переназначать открытые PR
- Обработка до 100 пользователей за < 100ms

### Тестирование (Интеграционное)
```bash
make integration-test
```
